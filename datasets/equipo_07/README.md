## Instrucciones

El dataset incluye atributos de varios dulces. Para las variables binarias, 1 significa sí y 0 significa no. Los datos contienen los siguientes campos:

+ `chocolate`: ¿Contiene chocolate?
+ `fruity`: ¿Tiene sabor a fruta?
+ `caramel`: ¿El dulce contiene caramelo?
+ `peanutalmondy`: ¿Contiene cacahuates, mantequilla de cacahuate o almendras?
+ `nougat`: ¿Contiene nougat/turrón?
+ `crispedricewafer`: ¿Contiene arroz inflado, obleas o algún componente tipo galleta?
+ `hard`: ¿Es un dulce duro?
+ `bar`: ¿Es una barra de dulce?
+ `pluribus`: ¿Es uno de varios dulces contenidos en una bolsa o caja?
+ `sugarpercent`: Percentil de azúcar dentro del conjunto de datos.
+ `pricepercent`: Percentil del precio unitario comparado con el resto del conjunto.
+ `winpercent`: Calificación del dulce
