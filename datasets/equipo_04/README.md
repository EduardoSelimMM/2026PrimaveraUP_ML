## Instrucciones

La tarea de predicción es un problema de clasificación de 3 clases (columna `Target`):

+ Dropout: El estudiante abandonó el curso antes de completarlo.
+ Enrolled: El estudiante sigue inscrito en la fecha normal de finalización del curso.
+ Graduate: El estudiante completó exitosamente el curso.

Comparar modelos basados en árboles y redes neuronales.

**IMPORTANTE:** Muchas variables aparecen codificadas como numéricas, pero NO lo son. Identifiquen cuáles e intenten dar el significado de cada variable. Sugerencia: Revisen los valores únicos de cada columna.
